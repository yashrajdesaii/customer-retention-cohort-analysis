# Customer Retention & Cohort Analysis

## Overview
This project analyzes transaction-level customer data to build cohort views by first activity period, track repeat activity over time, and summarize retention trends through dashboards and reporting.

## What this repo will include
- A cohort model built from transaction-level data
- Cohort retention tables and summary metrics
- Dashboards and reporting views (Tableau or Excel)
- Clear documentation and reproducible steps

## Current status
In progress. Repo structure and documentation are set up. Implementation will be added shortly.

## Tools
Python (pandas), SQL, Excel, Tableau

## Notes on data
Any included data will be synthetic or publicly available and used for portfolio purposes only.
